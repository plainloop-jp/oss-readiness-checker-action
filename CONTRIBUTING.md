# Contributing

Thanks for considering a contribution.

This repository is the GitHub Marketplace wrapper for
[OSS Readiness Checker](https://github.com/plainloop-jp/oss-readiness-checker).

## Where to make changes

- Open an issue or pull request here for changes to `action.yml`, `src/action.js`,
  or this wrapper's documentation.
- Use the [main repository](https://github.com/plainloop-jp/oss-readiness-checker)
  for changes to the checker rules or CLI output.

## Before opening a pull request

1. Keep the pull request focused on one improvement.
2. Run the action locally when changing `src/action.js`.
3. Explain how you checked the change.
